# Asfahas Barbershop

A modern and mobile-friendly website for Asfahas Barbershop in Skanstull, Stockholm.

Live site:  
https://asfahas-barbershop.netlify.app/

## About the project

This is a static website built to present the barbershop’s services, prices, opening hours, customer reviews, contact information and gallery.

The website does not include a custom booking system. Booking is handled through phone contact or Bokadirekt to keep the project simple, stable and easy to maintain.

## Features

- Responsive design for mobile and desktop
- Clear hero section with call-to-action buttons
- Gallery with haircut examples
- Services section
- Price list
- Google customer reviews
- Opening hours
- Contact information
- Instagram, Snapchat and Google Maps links

## Built with

- HTML
- CSS
- JavaScript
- GitHub
- Netlify

## Roadmap

### Phase 1: Static website
Build and publish the main website with design, content, contact information, prices, gallery and live deployment.

### Phase 2: Firebase gallery
Create a dynamic gallery where images can be uploaded and displayed automatically on the website.

### Phase 3: Admin panel
Build a private admin panel where the barber can log in and manage gallery images.

### Phase 4: AI caption generator
Allow the barber to upload an image and receive three Swedish Instagram caption suggestions generated by AI. The barber copies the text manually and posts it in the Instagram app.

The project will not connect directly to the Instagram API, because that would make the system more unstable and increase the risk of bugs.

### Phase 5: Polish and client demo
Final improvements, mobile testing, feedback and presentation to the client.
